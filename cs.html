CSS

CSS, que significa "Hojas de Estilo en Cascada" en inglés (Cascading Style Sheets), es un lenguaje de estilo utilizado para describir la presentación visual de un documento escrito en HTML o XML. Su principal función es separar la estructura y el contenido de un documento HTML de su diseño y apariencia.

En lugar de incluir el estilo directamente en el código HTML, CSS permite definir reglas de estilo en un archivo separado. Estas reglas especifican cómo deben presentarse los elementos HTML, como colores, fuentes, márgenes, tamaños de texto, etc.
